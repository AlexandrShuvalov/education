# Редактор базы заданий

Это отдельный локальный сайт для наполнения базы заданий. Он не подключается к рабочему сайту и не отображается в меню основного `index.html`.

## Как открыть

Откройте двойным кликом файл:

```text
task-editor/index.html
```

## Как пользоваться

1. Выберите экзамен: ОГЭ или ЕГЭ.
2. Выберите номер задания.
3. Выберите шаблон: выражение, задача с вариантами, задача с картинкой, утверждение или полное решение.
4. Заполните поля и проверьте предпросмотр.
5. Нажмите `Добавить в файл`.
6. Когда все задания добавлены, нажмите `Скачать обновленный файл`.
7. Скачанный файл `task-XX.js` положите вместо старого файла в папку `tasks` или `ege-tasks`.

## Картинки

Картинки храните в основной базе:

```text
tasks/images/task-07/
```

В редакторе указывайте путь без `tasks/`, например:

```text
images/task-07/task-07-022.png
```
